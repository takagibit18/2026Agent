### 1. Multi-Agent vs ReAct

- **Multi-Agent：** 多个智能体协作完成任务，每个 Agent 负责特定子任务（如 MergeWarden 的审查 Agent + 调试 Agent）
- **ReAct：** Reasoning + Acting 范式，Agent 通过"思考 → 行动 → 观察"循环解决问题
### 2. Function Call（工具调用）

LLM 通过结构化输出调用外部工具（如搜索、代码执行），你的 MergeWarden 项目中的"分级工具系统"就是典型应用。
### 3. RAG 核心流程

检索（Retrieval）→ 增强（Augmentation）→ 生成（Generation）。你的 RAG Lab 项目完整覆盖了这三个环节。
### 4. Prompt Engineering
通过优化提示词提升 LLM 输出质量，包括 Few-shot、Chain-of-Thought、结构化输出等技巧。
